# HearthStone-Manager
Gerenciador de cartas HearthStone utilizando ReactJS e Recoil


## Demo
https://wscld.github.io/HearthStone-Manager/

## Como Rodar
    $ yarn install
    $ yarn start

## Dependencias e APIs
- axios
- sweetalert2
- node-sass
- gh-pages
- Blizzard API (https://develop.battle.net/documentation/hearthstone/game-data-apis)
